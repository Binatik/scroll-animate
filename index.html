Анимация на чистом js

Использование анимации на чистом html,css,js, без каких-либо библиотек.

Простая анимация

HTML

<h2 class="example__title animations">Простая анимация </h2> 
<p class="example__text animations">class animations </p>

CSS

.example__title { 
    opacity: 0; 
    transform: translateY(-15px); 
    transition: 1.2s;
}  

.example__title.anim {  
    transform: translateY(0);
    opacity: 1;  
} 

.example__text {  
    transform: translateX(180px);
    transition: 1.2s;  
} 

.example__text.anim {  
    transform: translateX(0); 
}

Сложная анимация

HTML

<h2 class="child animations">Сложная анимация </h2> 
<p class="child animations">class animations 1-child </p> 
<p class="child animations">class animations 2-child </p> 
<p class="child animations">class animations 3-child </p>

CSS

.child { 
    opacity: 0; 
    transform: translateX(-150%); 
    transition: 0.8s;
}  

.child.anim:nth-child(1) {  
    transform: translateX(0); 
    transition: 0s;
    opacity: 1;  
} 

.child.anim:nth-child(2) {  
    transform: translateX(0); 
    transition: 1.3s;
    opacity: 1;  
} 

.child.anim:nth-child(3) {  
    transform: translateX(0); 
    transition: 1.6s;
    opacity: 1;  
}  

.child.anim:nth-child(4) {  
    transform: translateX(0); 
    transition: 1.9s;
    opacity: 1;  
}
